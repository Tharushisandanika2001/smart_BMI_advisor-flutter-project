# BMI Calculator App 

A fully functional, cross-platform application for calculating **Body Mass Index (BMI)** and providing **personalized health advice** based on user results. The project follows a **microservice-based architecture** with a Python backend handling business logic and a Dart frontend delivering a smooth user experience across mobile and web.

---

##  Features

*  **BMI Calculation** ‚Äì Calculates BMI using height and weight inputs
*  **Personalized Health Advice** ‚Äì Suggestions based on BMI category
*  **Cross-Platform UI** ‚Äì Works on web
*  **Microservice Architecture** ‚Äì Clear separation of frontend and backend

---

##  Tech Stack

**Frontend:** Dart (Flutter)
**Backend:** Python (Microservice-based API)
**Architecture:** RESTful API communication

---

##  Getting Started

### Run Backend

```bash
cd backend
python app1.py
```

### Run Frontend

```bash
cd my_web_app
flutter pub get
flutter run
```

##  Purpose

This project demonstrates the use of **Dart + Python** in a real-world application to build a scalable, maintainable health-related system using modern software architecture principles.
